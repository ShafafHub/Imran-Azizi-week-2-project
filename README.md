Pure Node.js REST API (no frameworks)
## what i build
A REST API called `products-service` that can:
- List products
- Get a product by id
- Create a product
- Update a product
- Delete a product
## Run

npm install  
npm start

## Endpoints

GET /products  
GET /products/:id  
POST /products  
PUT /products/:id  
DELETE /products/:id

## Example

```bash
	http://localhost:3000/products
```
