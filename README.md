# Products Service

Pure Node.js REST API (no frameworks)

## Run

npm install  
npm start

## Endpoints

GET /products  
GET /products/:id  
POST /products  
PUT /products/:id  
DELETE /products/:id

## Example

```bash
GET	http://localhost:3000/products

POST	http://localhost:3000/products

PUT	http://localhost:3000/products/1

DELETE	http://localhost:3000/products/1

```
